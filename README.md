# 📊 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

This project demonstrates an end-to-end **E-Commerce Sales Analytics Solution** using **Python, PostgreSQL, SQL, and Power BI**.

The objective was to analyze customer behavior, sales performance, product trends, payment patterns, and overall business revenue through data analysis and interactive dashboards.

# 🔄 Project Workflow

### 📂 Data Preparation

* Generated and prepared E-Commerce datasets using Python.
* Cleaned data using Pandas.
* Removed duplicates and handled missing values.
* Standardized categorical fields.

### 🗄️ Database Design

* Imported cleaned datasets into PostgreSQL.
* Designed relational database schema.
* Created Primary Key and Foreign Key relationships.

### 📝 SQL Analysis

* Performed business analysis using SQL.
* Applied Joins, Aggregate Functions, GROUP BY, Subqueries, CTEs, and Window Functions.
* Extracted meaningful business insights.

### 📊 Dashboard Development

* Connected PostgreSQL with Power BI.
* Built KPIs and interactive visualizations.
* Developed an executive-level sales analytics dashboard.
